An educational ocean exploration game where players control a curious squid discovering various marine life species. Perfect for children to learn about ocean creatures while having fun!

## Installation

1. Make sure you have Python 3.7+ installed
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## How to Play

Run the game using:
```
python main.py
```

- Use arrow keys or WASD to swim around
- Explore the ocean and discover different marine species
- Learn interesting facts about ocean creatures you encounter

## Features

- Smooth swimming controls
- Educational content about marine life
- Beautiful underwater environment
- Kid-friendly interface

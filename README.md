# Squid Bubbles

A charming underwater exploration game where you discover and learn about marine life. Play as either a curious shrimp or an inquisitive squid as you swim through the ocean, encountering and learning about various sea creatures.

## Game Features

- Play as either a Shrimp or Squid character
- Fluid swimming mechanics with realistic underwater movement
- Discover and learn about various marine species
- Beautiful underwater environment with dynamic effects
- Educational marine life database with interesting facts
- Unique movement patterns for different sea creatures

## Technical Details

- Built with Unity 2022.3 LTS
- 2D top-down perspective
- Utilizes Unity's Universal Render Pipeline (URP) for enhanced visuals
- Implements custom physics for underwater movement
- Features scriptable objects for easy content management

## Installation

1. Clone the repository
2. Open the project in Unity 2022.3 or later
3. Open the main scene in Assets/Scenes
4. Press Play to start the game

## Controls

- WASD or Arrow Keys: Swim
- Left Mouse Button/Space: Dash
- Mouse: Aim
- ESC: Pause Game

## Development Setup

### Requirements
- Unity 2022.3 LTS or later
- Basic Unity development environment

### Project Structure
- `/Assets/Scripts`: Core game scripts
- `/Assets/Prefabs`: Reusable game objects
- `/Assets/ScriptableObjects`: Marine life database and presets
- `/Assets/Scenes`: Game scenes
- `/Assets/Art`: Sprites and visual assets
- `/Assets/Audio`: Sound effects and music

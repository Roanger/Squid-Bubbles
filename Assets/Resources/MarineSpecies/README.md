## Placeholder Creation Steps
1. Create a new 256x256 image in your preferred image editor
2. Use simple geometric shapes for the basic form
3. Add basic details with a few colors
4. Export as PNG with transparency
5. Place in this folder

## Example Style Guide
- Main body: Solid fill with slight gradient
- Eyes: Simple white circles with black dots
- Details: Basic shapes for fins, tails, etc.
- Colors: Bright, vibrant colors suitable for children
- Outline: Optional thin outline for better visibility

## File Naming Convention
- Use camelCase naming
- No spaces in filenames
- Include species name only
- Example: blueWhale.png, clownfish.png

## Quick Start
For quick placeholder creation:
1. Use Unity's built-in sprite editor
2. Create basic shapes for each species
3. Use the color picker for child-friendly colors
4. Keep designs simple but recognizable
